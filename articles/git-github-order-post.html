如何用git开始github之旅

2016-03-15 19:40
By lidatong

下载并安装git，注册github账号

下载安装注册都很容易，如果在安装git时不知道如何选，一路默认也是可以的。

安装SSH key

要使用Github，必须先在本机上生成SSH key，并且将其添加到自己github账号的SSH Keys上。产生的方法是：

ssh-keygen -t rsa -b 4096 -C "your_email@example.com"

然后一路回车，结束后还需要将SSH key添加到ssh-agent，方法为：

ssh-add ~/.ssh/id_rsa

之后就是需要将id_rsa.pub中的内容添加到github账号的SSH keys上。

常用的git指令

通常新建一个repository时，在一台新的电脑上，往往都需要用到的相关指令：

在github上new了一个xxx.github.io.git库之后，可以：

git init
git add .
git commit -m "test"
git remote add origin git@github.com:xxx/xxx.github.io.git
git push -u origin master

如果是github上本来就存在了xxx.github.io.git，并有文件，可以：

git init
git remote add origin git@github.com:xxx/xxx.github.io.git
git pull
git add .
git commit -m "test"
git push -u origin master

最后一步如果不成功，比如出现错误如：

则使用强制push：

git push -f

一般都能够成功。
