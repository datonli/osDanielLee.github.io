使用python读取二进制字节流文件的方法

使用C的fwrite直接将char[]的内容写到txt文件中是一种很常见 的写文件方式（尤其是在下位机等较为底层的应用场景），而我们 往往使用python读取文件更多的是考虑读取文件中的内容为字符形式 （包括字符串和数字），遇到这种一个一个char的情况就觉得非常棘手。

其实在python中read(1)就是代表一个一个char来读取的作用，例子如下：

with open("QQQ6.TXT", "rb") as f:
    byte = f.read(1)
    byteList = list()
    while byte:
        ob = ord(byte)
        byteList.append(ob)
        byte = f.read(1)
    print(byteList)
    print(len(byteList)/64)
for i in range(5):
    l = byteList[i*64:(i+1)*64]
    print(l)
    print(len(l))
